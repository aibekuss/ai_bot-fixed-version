## Настройка за 5 минут

### 1. Google Credentials

Скачайте ключ сервисного аккаунта из Google Cloud Console и сохраните как:
```
ai_bot/nomad-whatsapp-bot/google-credentials.json
```

### 2. Переменные окружения

Создайте или обновите `.env`:

```bash
# Google Sheets
GOOGLE_SHEETS_ENABLED=true
GOOGLE_SHEETS_ID=ваш_id_таблицы

# Google Calendar
GOOGLE_CALENDAR_ENABLED=true
GOOGLE_CALENDAR_ID=primary

# Путь к ключу
GOOGLE_APPLICATION_CREDENTIALS=./google-credentials.json
```

### 3. Предоставить доступ

**Google Таблица:**
- Открыть таблицу → Поделиться
- Добавить email из google-credentials.json
- Права: **Редактор**

**Google Calendar:**
- Открыть Calendar → Настройки календаря
- Доступ для конкретных пользователей
- Добавить email сервисного аккаунта
- Права: **Вносить изменения в события**

### 4. Запуск

```bash
cd ai_bot/nomad-whatsapp-bot
npm install
npm run build
npm start
```

### 5. Проверка

В логах должно быть:
```
✅ Google Sheets подключен
✅ Google Calendar подключен
```
