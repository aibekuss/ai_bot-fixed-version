# 🚀 Запуск бота с Google Sheets

## Быстрый старт (без Google Sheets)

```bash
cd ai_bot/nomad-whatsapp-bot
rm -rf auth_info          # Удалить старые сессии
npm start                 # Запустить бота
```

📱 Отсканируйте QR-код в WhatsApp

---

## С Google Sheets (для записи пациентов)

### 1. Настройка Google Sheets

Следуйте инструкции в [GOOGLE_SHEETS_SETUP.md](./GOOGLE_SHEETS_SETUP.md)

Кратко:
1. Создать проект в Google Cloud
2. Включить Google Sheets API
3. Создать сервисный аккаунт и скачать JSON-ключ
4. Создать таблицу и дать доступ сервисному аккаунту
5. Положить ключ в `google-credentials.json`
6. Добавить в `.env`:
   ```env
   GOOGLE_SHEETS_ENABLED=true
   GOOGLE_SHEETS_ID=ваш_ID_таблицы
   ```

### 2. Запуск

```bash
cd ai_bot/nomad-whatsapp-bot
rm -rf auth_info
npm start
```

### 3. Проверка

В логе должно быть:
```
✅ Google Sheets подключен
✅ Заголовки таблицы созданы
```

При записи пациента в Google Таблицу автоматически добавится строка:
- ID записи
- Имя пациента
- Телефон
- Возраст
- Услуга
- Дата
- Время
- Дата создания
- Статус

## 🔧 Команды

```bash
npm install      # Установить зависимости
npm run build    # Скомпилировать TypeScript
npm start        # Запустить бота
npm run qr       # Показать QR-код (если нужно)
```

---

## ⚠️ Важно

- Файл `google-credentials.json` **не коммитьте в Git** (он в `.gitignore`)
- При ошибке 401 удалите папку `auth_info` и отсканируйте QR заново
- Бот игнорирует группы WhatsApp (отвечает только в личных сообщениях)
