## 💡 Как добавить новые данные:

Просто добавьте новый JSON файл в `src/data/`:
```json
{
    "faq": [
        {
            "question": "Нужна ли запись?",
            "answer": "Да, запись обязательна"
        }
    ]
}
```

Бот автоматически загрузит его при следующем старте!
